# Data licensing

OpenWeer's source code is released under the [MIT License](./LICENSE).

The **weather data** displayed and served by OpenWeer is provided by the **Royal Netherlands Meteorological Institute (KNMI)** through its [Open Data Platform](https://dataplatform.knmi.nl/) and is licensed under [Creative Commons Attribution 4.0 (CC-BY-4.0)](https://creativecommons.org/licenses/by/4.0/).

If you reuse OpenWeer's outputs (radar tiles, forecast values, observation data) you **must**:

1. Attribute KNMI as the source. The recommended attribution string is:
   > Weather data © KNMI, [CC-BY-4.0](https://creativecommons.org/licenses/by/4.0/).
2. Indicate if you made changes to the data.
3. Not apply legal terms or technological measures that legally restrict others from doing anything the licence permits.

OpenWeer code does not relicense the upstream data; it merely processes it.
